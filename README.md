Fungsi dari porgram yang saya buat adalah untuk menhitung volume dari sebuah tabung, yang jari jari dan tinggi tabungnya di input oleh user.
adapun cara kerja program ini adalah 
1. User menginput jari jari dan tinggi dari sebuah tabung yang ingin dihitung;
2. program akan menjalankan rumus yang sudah di buat menggunakan konsep fungsi;
3. program mengeluarkan output berupa hasil perhitungan volume tabung.
